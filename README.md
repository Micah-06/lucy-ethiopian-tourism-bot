# Lucy Ethiopian Tourism Bot 🇪🇹

Lucy is an AI-powered tourism assistant designed to help travelers explore Ethiopia.

## Features
- Ethiopian attractions guide
- Hotels and restaurants suggestions
- Transportation information
- Visa guidance
- Emergency contacts
- Culture and food recommendations
- Multi-language support (coming soon)

## Tech Stack
- n8n
- Node.js 
- Google Sheets (knowledge base)
- AI chatbot integration
- API-based response system

## Knowledge Base
Stored in Google Sheets:
- Attractions
- Hotels
- Restaurants
- Transport
- Visa information
- Emergency contacts
